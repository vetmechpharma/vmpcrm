**what currently exists?**
A full-stack CRM application using FastAPI (Python) for the backend, React for the frontend, and MongoDB as the database. The application provides:
- JWT authentication for admin users.
- Full CRUD functionality for doctors, products, and transport providers.
- A public, mobile-responsive ordering page with WhatsApp OTP verification.
- An admin dashboard with statistics and management panels for orders, company/API settings, pending items, and leads (doctors).
- A sophisticated order management workflow with status-based conditional UI and automated WhatsApp notifications.
- A complete lead follow-up system with notes, tasks, and reminders.
- A system for managing out-of-stock items by moving them to a per-customer pending list.
